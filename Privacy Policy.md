# Privacy Policy

**Effective Date:** 04/22/2025  
**Last Updated:** 02/02/2026

Your privacy is important to us. This Privacy Policy outlines how we collect, use, and protect your information when you use **Gag bot**.

## 1. Information We Collect

We limit our data collection to the minimum necessary for the App to function:
* **Core Account Data:** Your Discord user ID.
* **Error Logs:** Command usage and arguments provided during a system error or crash.
* **Anonymized Metrics:** Aggregated usage statistics (e.g., total command volume, server counts) which do not identify individual users.
* **Discord API Data:** To execute commands, the App accesses standard data provided by the Discord API (such as server roles, permissions, and usernames). This data is generally used in real-time and is not permanently stored unless specifically noted.

## 2. Message Handling

Messages are processed **in real time** to provide functionality (e.g., responding to commands or performing actions).
* **No Storage of Content:** We do not store or log general message content. Once processed, the message is immediately discarded.
* **Error Exceptions:** If a command fails, the specific command and its arguments may be temporarily logged to help our developers fix the bug.
* **Anonymized Statistics:** We may log that a command was used and at what time for performance monitoring. This does not include user-identifying data.
